# Flagship workflows

Forge’s first-class page types are designed to feel like one product, not three disconnected tools.

## Contact & booking

**Best for:** local services, consultants, anyone who needs leads and optional scheduling.

- Studio primes: “A contact form for my business — …”
- Page Detail emphasizes submissions and calendar automations.
- Dashboard filter: **Contact forms**.

## Proposals & quotes

**Best for:** contractors, agencies, anyone sending priced scopes.

- Studio primes: “A proposal for …”
- Page Detail labels submissions as **Viewers & decisions**; pipeline context lives on Overview.
- Dashboard filter: **Proposals**.

## Pitch decks

**Best for:** founders, PMs, internal strategy readouts.

- Studio primes: “A pitch deck for …”
- Page Detail adds **Present**, **Export** (PDF/PPTX queue), and **Viewers** analytics.
- Dashboard filter: **Decks**.

## Cross-links

- **Templates** gallery opens with workflow-first cards (search `q=` shortcuts).
- **Onboarding** can store `onboarded_for_workflow` and deep-link into Studio.
- **Command palette** includes workflow shortcuts; keyboard: `Ctrl/⌘+Shift+C/P/D`.

When intent is ambiguous, Studio may show a non-blocking **workflow clarify** chip row; generation continues with the default while you optionally pick a different branch.

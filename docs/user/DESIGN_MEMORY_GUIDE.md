# Design Memory — user guide

Design Memory is Forge’s way of remembering stylistic and workflow preferences you signal with thumbs, quick actions, and refinements.

## Where to manage it

- **Settings → Memory** — list entries, see strength, reset all.
- **Settings → Privacy** — cross-workspace memory + whether anonymized feedback can inform platform improvements.

## What “strength” means

Each entry has a score from 0–1. Higher values influence future generations more strongly. Low-strength rows may be pruned automatically by the nightly learning loop.

## Fresh start

Use **Forget everything** if you want Forge to stop biasing toward past experiments. You can still leave feedback — new memory will accumulate from scratch.

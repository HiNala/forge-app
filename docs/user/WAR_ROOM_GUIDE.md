# War Room guide (users)

Forge’s **Product War Room** organizes work into three columns:

| Pane | Purpose |
| --- | --- |
| **Strategy** | Goal, personas, layered reasoning, memory bullets (BP-02). |
| **Canvas** | Live HTML preview; responsive truth; marquee workflows shared with Forge canvases. |
| **System** | Emerging Layer 2 structuring—JSON until tree editors stabilize. |

Under the preview: **Next move** (single recommendation) and the **Action dock** (deploy, simulate scaffold, shortcuts).

Stages (`?stage=`): **`idea · system · design · build · grow`** — reflow pane emphasis (`IDEA` tucks away System initially).

Revert to classic Studio: **Settings → Studio**, or **`/studio?legacy=1`**.

Keyboard: **`⌘/`** (**Ctrl+/**) focuses Strategy goal when War Room owns focus handlers.
